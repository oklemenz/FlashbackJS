# Flashback

Conrad's animation in Flashback

## Play GitHub Version

- Browser: https://oklemenz.github.io/FlashbackJS

## Play Locally

- Install [Node.js](https://nodejs.org)
- Clone: `https://github.com/oklemenz/FlashbackJS.git`
- Terminal:
  - `npm install`
  - `npm start`
- Browser: `localhost:8080`
